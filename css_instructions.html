Introduction to CSS

The CSS box model

There are four parts to the CSS box model. The first is the content, this is what the person viewing your webpage will see. The second is the padding, this is what's between your content and the container it is in. The third is the border, this is an optional thing that you can add to really separate your content from other content in your page. The fourth is margin, this is similar to padding but is the space at the end of all of this and pushes other content away from the content in your box.

Positioning

When you position things using the position property things will stay where you place them.

This is what happens with the relative property. The element is positioned relative to its normal position, so "left:20px" adds 20 pixels to the element's LEFT position

This is what happens with absolute property.The element is positioned relative to its first positioned (not static) ancestor element

This is what happens with fixed property. The element is positioned relative to the browser window

Shadows

The h-shadow property sets the position of the horizontal shadow

The v-shadow property sets the position of the vertical shadow.

The blur-radius property sets the blur amount of the shadow.
